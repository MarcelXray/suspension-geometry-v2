# Fahrwerksgeometrie 1:10 Tourenwagen

Interaktiver Doppelquerlenker-Geometrie-Visualisierer für 1:10 RC-Tourenwagen.

🔗 **[Live-Demo](https://marcelxray.github.io/suspension-geometry/)**

## Features
- Kinematische Berechnung (Rollzentrum, Sturz, Momentanpol)
- Wanken & Einfedern Animation
- Hell/Dunkel-Design
- Setup speichern/laden (JSON)
- Zoom (Mausrad oder Slider)
- Mobil-optimiert

## Lokale Entwicklung
```bash
npm install
npm run dev
npm run build
```

## Deployment
Automatisch per GitHub Actions bei jedem Push auf `main`.
GitHub Pages: `gh-pages` Branch, `/ (root)`.
